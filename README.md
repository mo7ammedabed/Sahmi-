# سهمي 🇵🇸 - منصة التمويل الجماعي الفلسطينية

سهمي هي أول منصة تمويل جماعي فلسطينية 100%  
نساعد أصحاب المشاريع والأفكار في فلسطين على جمع التمويل من المستثمرين في كل العالم.

## المميزات
- إضافة مشاريع بسهولة (لأصحاب المشاريع)
- استثمار آمن وسريع (للمستثمرين)
- لوحة تحكم خاصة لكل نوع مستخدم (مستثمر / صاحب مشروع / أدمن)
- دعم رفع الصور ومتابعة نسبة التقدم
- تصميم متجاوب ومتوافق مع الموبايل
