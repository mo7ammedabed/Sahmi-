# Sahmi-
🇵🇸 سهمي - منصة التمويل الجماعي الفلسطينية الذكية | Sahmi - Palestinian Crowdfunding Platform  منصة إلكترونية آمنة تربط أصحاب المشاريع الطموحين بالمستثمرين الأذكياء. نوفر بيئة شفافة للاستثمار الجزئي، مع دعم كامل للغة العربية (RTL)، نظام أدوار متكامل (مستثمر، صاحب مشروع، أدمن)، وخوارزمية آلية لتوزيع الأرباح. مبني باستخدام Django 5.2 و PostgreSQL
